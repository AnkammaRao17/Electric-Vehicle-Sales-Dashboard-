# ⚡ Electric Vehicle Sales Dashboard (Excel)

## 📌 Overview
This project presents an interactive **Electric Vehicle (EV) Sales Dashboard** created using Microsoft Excel.  
The dashboard provides insights into EV sales trends across different states, years, and vehicle categories.

---

## 📊 Dashboard Features
- Total Sales Overview
- Average Sales Metrics
- Year-wise Sales Analysis
- Month-wise Sales Trends
- State-wise Sales Comparison
- Vehicle Category Distribution
- Month-on-Month (MoM) Change
- Year-on-Year (YoY) Growth

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers (Filters)
- Data Visualization

---

## 📁 Files Included
- `EV_Dashboard.xlsx` → Main Excel dashboard file
- `ev.csv` (optional) → Raw dataset

---

## 🎯 Key Insights
- Identifies top-performing states in EV sales
- Tracks growth trends over time
- Compares different vehicle categories
- Highlights seasonal sales patterns

---

## ▶️ How to Use
1. Download the Excel file
2. Open in Microsoft Excel (recommended: Excel 2016 or later)
3. Use slicers to filter:
   - Year
   - Month
   - State
4. Interact with charts for insights
